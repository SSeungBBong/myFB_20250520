# myFB_20250520
미니페북을 정리해서 간결하게 만드는 과정

## 프로그램 설명 
1. 이 프로그램은 창업트랙 수업에서 실습과정으로 만든것임
2. 사용된 기술은 다음과 같음   
   . pyQT - GUI를 구현하기 위함   
   . QTDesigner - GUI를 디자인 하기 위함   
   . pyInstaller - 1파일 exe, 파일과 nonConsole 옵션을 지정 했다. -w -F 옵션을 사용   
   . installFactory - 설치 파일을 만들기 위함.
